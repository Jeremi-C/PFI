Pauper et les cartes d'un deck
Pour jouer a Magic The Gathering il faut un deck. Un deck est un groupe de cartes Magic qui consiste d'environ un tier de carte terrain et deux tiers carte non-terrains. Pauper est un type de format qui restraint le type de carte qui peuvent être utiliser. Les seuls cartes sont les cartes communes. Il est utile pour avoir un deck qui coute moins chère, car ceux si sont les plus commune.
